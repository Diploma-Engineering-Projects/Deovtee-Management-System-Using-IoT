# Devotee Management System Using IoT 
This project develops a Devotee Management System for temple darshan using an Arduino. It uses a pair of ultrasonic sensors to detect and count devotees entering and exiting the temple. The Arduino processes the sensor signals, updates the visitor count, and calculates the number of devotees currently inside the temple. A 16×2 LCD displays the entry count, exit count, and current occupancy. The system continuously monitors the sensors and operates through embedded programming stored in the microcontroller.
## Requirement 

### Software :
* Arduino IDE
* ThindSpeak IoT Cloud Platform
### Hardware :
* Arduino UNO Board
* Ultrasonic Sensor
* LCD Display 16 x 2 
* Buzzer
## Usage
Devotee management systems have emerged as essential tools for religious
organizations to effectively manage and engage their devotees. It emphasizes the need for
organizations to proactively address challenges and adapt to emerging trends to ensure
successful implementation and long-term engagement with devotees.

## Future Enhancements 
The counter system can detect two simultaneous visitors, entering or leaving at the same
time. The system is developed using Arduino UNO development board and its IDE. The project
uses two ultrasonic sensors, LCD 16x2, Arduino Uno Board. The system is designed to use at
places where the intensity of crowd is intermediate. The system can be enhanced by using the
camera and image processing which have better sensitivity at detecting persons at high
accuracy. The size of the circuit can also be increased based on the area available. The data can
be sent to cloud platform and store it for future use.
